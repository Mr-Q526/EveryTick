# 万物打卡 EveryTick

万物皆可打卡 —— 为任何事情创建专属的打卡事件，养成习惯、记录坚持。

Create check-in events for anything. Build habits, track streaks, celebrate consistency.

## ✨ 核心理念

生活中有太多值得被记录的事：坐了一次飞机、吃了一顿火锅、去了一次健身房、读完了一本书……

**万物打卡**让你为任何事情创建一个打卡项目，每次完成时轻点一下 **+1**。不需要复杂的设置，不需要联网，打开即用。

> 每一次，都值得被记住。

## 📱 功能亮点

### 🎯 自由创建打卡项目

- 为**任何事情**建立打卡事件 —— 坐飞机、吃火锅、读书、跑步……
- 自定义**项目名称**和**主题颜色**（8 种精选配色）
- 内置快速模板：✈️ 起飞、🍲 美食、💪 健身，一键套用

### 📝 灵活的自定义字段

每个打卡项目可以附加多种字段类型，记录你关心的细节：

| 字段类型 | 用途 | 示例 |
|---------|------|------|
| 📍 分类/地点 | 记录场景或位置，自动生成排行榜 | 店名、城市、训练部位 |
| 💰 货币金额 | 追踪花费 | 消费金额（¥） |
| ⏱ 花费时长 | 记录时间消耗 | 训练时间（分钟） |
| 🔘 单选项 | 从预设选项中选一个 | 训练强度（低/中/高）、心情（好/一般/差） |
| ☑️ 多选项 | 从预设选项中选多个 | 训练部位（胸/背/腿/肩）、读书标签 |
| ✅ 是否 | 开关式布尔记录 | 是否破纪录、是否空腹 |
| 🏷️ 标签数值 | 多选标签 + 逐项输入数值 | 训练部位×时长、支出分类×金额、学科×时间 |
| 📓 长篇笔记 | 自由文本记录 | 心得体会 |
| 🔢 普通数值 | 通用数字 | 公里数、页数 |

### 📊 强大的数据分析

打卡不仅仅是 +1，还有**全维度数据洞察**：

- **频率趋势** — 按日/月柱状图呈现打卡频率
- **打卡热力图** — 类似 GitHub 贡献图，直观展示打卡密度（最近 15 周）
- **星期分布** — 发现你在周几最活跃
- **时段分布** — 了解凌晨、上午、下午、晚上的打卡习惯
- **字段聚合** — 自动计算累计总计、平均每次、单次最高/最低
- **分类排行** — 分类/地点/单选/多选字段自动生成 Top 排行
- **是否比例** — 布尔字段展示百分比和比例条
- **标签数值排行** — 每个标签独立统计出现次数和数值总和
- **多维透视分析** — 按分类字段交叉分析数值字段（如：按「餐厅」统计「花费」）
- **时间筛选** — 支持 1 周 / 1 月 / 3 月 / 1 年 / 全部多维度查看

### ⏪ 补打卡

忘记打卡了？没关系！在打卡时点击「打卡时间」即可选择过去的日期和时间，补录历史记录。

### 🔒 隐私与本地优先

- **数据完全存储在本地**，使用 SharedPreferences 持久化 JSON 数据
- **无需注册、无需联网**，打开即用
- 你的数据只属于你

## 📂 项目结构

```
lib/
├── main.dart              # 应用入口 & 路由配置
├── constants/
│   └── presets.dart        # 内置快速模板（起飞、美食、健身）
├── models/
│   └── models.dart         # 数据模型（EventTemplate / EventRecord / FieldDefinition）
├── providers/
│   └── data_provider.dart  # 全局状态管理（InheritedWidget）
├── screens/
│   ├── home_screen.dart    # 首页仪表盘（项目卡片网格）
│   ├── new_event_screen.dart   # 新建项目（模板 + 自定义字段）
│   ├── record_screen.dart      # 打卡记录（填写字段值）
│   └── analytics_screen.dart   # 数据分析（图表 + 热力图 + 透视表）
├── services/
│   └── storage_service.dart    # 本地持久化层
└── theme/
    └── app_theme.dart      # 设计令牌（颜色 / 圆角 / 阴影）
```

## 🤝 贡献指南

欢迎所有人参与完善万物打卡！无论是新增打卡模板、优化 UI 还是修复 Bug，我们都非常欢迎。

### 如何贡献

1. **Fork** 本仓库
2. 创建你的功能分支 `git checkout -b feature/awesome-preset`
3. 提交更改 `git commit -m 'feat: 添加读书打卡模板'`
4. 推送分支 `git push origin feature/awesome-preset`
5. 创建 **Pull Request**

### 贡献方向

- 🆕 **新增打卡模板** — 在 `lib/constants/presets.dart` 中添加更多预设模板（如：读书、喝水、理财、旅行等）
- 🎨 **UI 优化** — 改进交互体验、添加动画、适配更多屏幕尺寸
- 🌍 **国际化** — 添加英文等多语言支持
- 📤 **数据导出** — 支持 CSV / JSON 导出打卡数据
- 🔔 **提醒功能** — 本地通知提醒打卡
- 📊 **更多图表** — 丰富数据分析维度

### 提交规范

- `feat:` 新功能
- `fix:` 修复 Bug
- `docs:` 文档更新
- `style:` 样式调整
- `refactor:` 重构
